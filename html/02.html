'Date types'

Типы данных

В Javascript есть несколько типов данных:

Числа

Числа могут быть целыми (42, -5) и дробными (12.67, -0.2). 

const age = 37;
const x = -0.25;

Строки

Строки используются для текстовой информации.

const name = 'Misha';
const message = 'Hello, world!';

Boolean

Булев (логический) тип состоит из 2 значений true и false. Такие значения используются для проверки условий в конструкции if и циклах.

const isMarried = true;
const hasACat = false;

null и undefined

Также есть ещё два пустых значения:

  • null мы используем, когда переменная пока пустая, но позже будет содержать какое-то значение
  • undefined - это неопределённое значение, чаще всего получается в результате ошибки в программе.
// Если у тебя пока нет партнёра, но в принципе может быть
let partner = null;

// Тут undefined, поскольку значение не определено
let value;

Проверка типа

Оператор typeof позволяет узнать тип данных переменной или целого выражения.

Тип - это строка 'number', 'string', 'boolean', 'undefined' для простых типов либо 'object', 'function', 'bigint', 'symbol' для более сложных, которые мы изучим позже.

По историческим причинам typeof null даёт строку 'object'.

const name = 'Misha';
console.log(typeof name); // 'string'

const age = 37;
console.log(typeof (age + 1)); // 'number'

const isMarried = true;
console.log(typeof isMarried); // 'boolean'

console.log(typeof undefined); // 'undefined'
console.log(typeof null); // 'object'
